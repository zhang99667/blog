# Social card fonts

These WOFF files are deterministic build inputs for article social cards. They are not shipped to the browser.

- Family: Noto Sans SC
- Weight: 800
- Source package: `@fontsource/noto-sans-sc@5.2.9`
- License: SIL Open Font License 1.1, included in `LICENSE`
- Simplified Chinese SHA-256: `dcb2e590d4ec4d6dee1004fcd333990ae5941511459c4d2a3238706689844826`
- Latin SHA-256: `6c462a676276dfb8987aaa9c6c332e58dbdd1b4e7d8fda9761e6a3a0adcc1865`

The renderer verifies both checksums before generating any image. A font update must change the checksums and article image URLs together.
